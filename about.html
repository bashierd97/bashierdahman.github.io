About This Project

Welcome to GIF Showcase

This website was created to demonstrate how to build a clean, multi-page website using GitHub Pages that showcases GIF collections organized by different topics.

How It Works

Each page features a collection of GIFs related to a specific theme. The responsive design ensures the site looks good on all devices.

Adding Your Own GIFs

To customize this site with your own GIFs:

  1. Add your GIF files to the images folder
  2. Update the HTML files to reference your GIFs
  3. Customize the descriptions as needed
